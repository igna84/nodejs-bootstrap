# Node.js 서버 개발을 위한 Bootstrap

본 소스는 Node.js Express를 이용하여 Restful API 서버를 만들기 위한 기본 형태를 만드는 것에 목표를 두고 있다.

### 제공 기능
- 웹서버(Express)
- 로깅(Winston Library)
- MySQL DB 연동(Sequelize, MySQL2)
- CORS(cors)
- BodyParser(body-parser)
- JWT 로그인(Passport)
- File Upload/Download(Multer)
- File Read, Write
- 기본적인 정규식 제공
